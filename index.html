Quick Setup Guide

  1. Create a Discord Application
    • Go to the Discord Developer Portal
    • Click "New Application" and give it a name
    • Navigate to the "Bot" tab and click "Add Bot"
  2. Configure Bot Permissions
    • Go to the "OAuth2" tab
    • Select "URL Generator"
    • Choose required scopes: bot and applications.commands
    • Select bot permissions: Send Messages and Use Slash Commands
  3. Install Dependencies
    npm install
  4. Configure Environment
    # Create .env file DISCORD_TOKEN=your_bot_token MONGODB_URI=your_mongodb_uri ADMIN_USER_ID=your_discord_id CLIENT_ID=your_client_id CURRENCY_NAME=Vladbucks DEFAULT_CREDITS=10 PACK_COST=5 EARN_COOLDOWN=43200000 SPECIAL_CHANCE=0.1 LEGENDARY_CHANCE=0.01
  5. Deploy Commands and Start Bot
    npm run deploy npm start

Available Commands

/tcg open

Open a new pack of cards

/tcg showcollection

View your card collection

/tcg earn

Earn daily currency

/tcg tradeup

Trade 5 cards for a higher rarity

/tcg trade offer

Offer a trade to another user

/tcg battle

Challenge another user to a card battle

/tcg profile

View your collection statistics

/tcg help

List all available commands
