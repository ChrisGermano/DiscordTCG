DiscordTCG

↓

Discord Trading Card Game

Transform your Discord community into an engaging collectible card game experience. Create, trade, and battle with unique cards representing your community members.

Collect & Trade

Open packs to discover new cards, trade with friends, and build your collection. Each card is unique and represents a member of your community.

Battle System

Challenge other players to card battles using your collection. Strategic gameplay with unique card abilities and community-specific mechanics.

Community Integration

Cards are automatically generated from your server's members, creating a personalized experience that celebrates your community.

Commands & Features

open

Open a card pack to discover new cards for your collection.

showcollection

View your current card collection and manage your cards.

earn

Earn currency to purchase card packs and trade with others.

tradeup

Trade in multiple cards to receive a higher rarity card.

battle

Challenge other players to card battles using your collection.

fuse

Combine cards to create unique, powerful combinations.

trade

Trade cards with other players in your community.

profile

View your profile, stats, and collection highlights.

Installation

1

Prerequisites

Before starting, ensure you have:

  • Node.js 16.x or higher
  • npm (comes with Node.js)
  • A Discord account with server management permissions
  • A MongoDB Atlas account
  • A server with PM2 installed (for production)
2

Discord Bot Setup

Create and configure your Discord bot:

  1. Go to the Discord Developer Portal
  2. Click "New Application" and give it a name (e.g., "DiscordTCG")
  3. In the "Bot" section:
    • Click "Add Bot"
    • Enable "Message Content Intent" under Privileged Gateway Intents
    • Enable "Server Members Intent"
    • Save your changes
  4. In the "OAuth2" → "URL Generator" section:
    • Under "Scopes", select:
      • bot
      • applications.commands
    • Under "Bot Permissions", select:
      • Send Messages
      • Embed Links
      • Attach Files
      • Read Message History
      • Use Slash Commands
      • Add Reactions
      • Read Messages/View Channels
    • Copy the generated URL at the bottom
  5. Use the generated URL to invite the bot to your server
  6. Copy your bot token from the "Bot" section (click "Reset Token" if needed)
  7. Note your Application ID (Client ID) from the "General Information" section

Keep your bot token secure and never share it. If compromised, reset it immediately in the Developer Portal.

3

Clone & Setup

Get the code and install dependencies:

git clone https://github.com/yourusername/DiscordTCG.git cd DiscordTCG npm install
4

MongoDB Setup

Set up your database:

  1. Create a MongoDB Atlas account at mongodb.com/cloud/atlas
  2. Create a new cluster (free tier is sufficient)
  3. Create a database user with read/write permissions
  4. Add your IP address to the IP whitelist
  5. Get your connection string from the "Connect" button
5

Environment Configuration

Create a .env file in the root directory with all required variables:

# Discord Configuration DISCORD_TOKEN=your_bot_token_here # From Bot section CLIENT_ID=your_client_id_here # From General Information GUILD_ID=your_guild_id_here # Right-click server → Copy ID (Developer Mode must be on) # MongoDB Configuration MONGODB_URI=your_mongodb_connection_string_here DB_NAME=discordtcg # Bot Configuration CURRENCY_NAME=coins PACK_COST=100 DAILY_AMOUNT=50 MAX_CARDS_PER_PACK=5 # Optional: Production Settings NODE_ENV=production LOG_LEVEL=info

Replace all placeholder values with your actual configuration. The MongoDB URI should look like: mongodb+srv://username:password@cluster.mongodb.net/discordtcg

6

Deploy Commands

Register the slash commands with Discord:

node src/scripts/deployCommands.js

This only needs to be done once, or when you update command definitions. Make sure your bot has the applications.commands scope and proper permissions.

7

Development Testing

For local development and testing:

npm run dev

This runs the bot with nodemon for automatic restarts during development. Verify the bot is online and commands are working in your server.

8

Production Deployment

For production deployment using PM2:

# Install PM2 globally if not already installed npm install -g pm2 # Start the bot with PM2 pm2 start src/index.js --name discordtcg # Other useful PM2 commands: pm2 logs discordtcg # View logs pm2 monit # Monitor the bot pm2 restart discordtcg # Restart the bot pm2 stop discordtcg # Stop the bot pm2 delete discordtcg # Remove from PM2 # Set up PM2 to start on system boot pm2 startup pm2 save

PM2 will automatically restart the bot if it crashes and can start it on system boot. Monitor the logs for any permission or connection issues.
